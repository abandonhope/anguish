Automatic upgrade of legacy UNIX commands

Anguish knows about new cool replacements of legacy UNIX tools and will install them automatically when you try to use them.

This video shows someone attempting to use the legacy UNIX `ls` command. Anguish seemlesly downloads and install the superior npm-base "uber-cow-ls" which then runs instead.
